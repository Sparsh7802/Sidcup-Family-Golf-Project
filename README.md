# Sidcup Family Golf - Modern Web Landing Page

A visually immersive and highly interactive landing page for Sidcup Family Golf, built to showcase advanced frontend development techniques and smooth user experiences.

## 🚀 Key Features
*   *Dynamic GSAP Animations:* Implemented scroll-triggered animations and smooth transitions using the GSAP library.
*   *Custom Interactive Cursor:* Designed a custom cursor follower and hover effects for a "premium" feel.
*   *Responsive Design:* Fully optimized for mobile, tablet, and desktop viewports using CSS Flexbox and Grid.
*   *Modern UI/UX:* High-fidelity recreation of a professional sports facility landing page.

## 🛠️ Tech Stack
*   *Frontend:* HTML5, CSS3 (Custom Variables, Keyframes)
*   *Interactivity:* JavaScript (ES6+)
*   *Libraries:* GSAP (GreenSock Animation Platform)

## 📂 Project Structure
*   index.html: Main structure and SEO-friendly markup.
*   style.css: Custom styling and layout logic.
*   script.js: GSAP animation logic and interactive elements.
